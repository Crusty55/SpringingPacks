Pitch

Concept

Springing Packs is a platform game made on mobile and web. This game was created in Game Maker. Concept of the game is a package with a spring, trying to get “delivered” to its destination. Springing Pack is a 2D game that shows obstacles and enemy’s that try to deter you from reaching the goal point.

Genre

Springing Pack is a 2D platformer, made for people who have a little time to kill or want to experience a new type of platformer.

Reference

One game that helped us make Springing Packs is called “Hot Air” and it has basically the same mechanics that Springing Pack has.

Target

This game is target to anyone that wants to enjoy a fun/challenging experience on mobile or desktop.

Platform

Springing Packs is accessible through your phone or our Github Website!

Competitors

Our competitors in the 2D space, “Jetpack Joyride”, “Doodle Jump” etc.

Key Features

Springing Pack has 2 ways to interact with the game and that is touch screen or spacebar and the arrow keys. Spring Pack also has colorful levels that might sometimes lag if playing on mobile. Concept of the game is pretty easy to grasp.
